# IMDb Sentiment Analysis using BERT

## Overview

This project performs sentiment classification on IMDb movie reviews using a fine-tuned BERT model from Hugging Face Transformers.

## Dataset

- IMDb Movie Reviews Dataset
- 25,000 training reviews
- 25,000 testing reviews

## Model

- BERT Base Uncased
- Hugging Face Transformers
- PyTorch Backend

## Features

- Text Tokenization
- Fine-Tuning BERT
- Sentiment Prediction
- Accuracy Evaluation

## Technologies Used

- Python
- Transformers
- Datasets
- PyTorch
- NumPy

## Example Prediction

Input:

This movie was fantastic

Output:

Positive
